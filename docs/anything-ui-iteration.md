# Anything UI iteration (Phase 3)

Use Anything **Build** only for batches marked worthwhile in the planner output—or one screen at a time after a Cursor batch ships.

## Workflow

1. Run Learn v2 locally or on Vercel; screenshot the screen you are changing.
2. Paste screenshot into Anything; prompt: match dark slate + indigo accent; one screen only; mock data; no auth/backend.
3. At most **two** iteration prompts per screen.
4. Note **keep / change / drop** (three bullets).
5. Implement changes in Cursor as the next `BATCHES.md` batch; do not import Anything’s generated repo.

## Skip credits when

- The planner said “go straight to code.”
- You are tweaking copy or logic only.
- Anything would rebuild the whole SAT or Today flow.

See [anything-planner-prompt.md](./anything-planner-prompt.md) for Phase 1 planning.
