# Roadmap

Learn v2 is currently a public-beta candidate. The goal is to keep the app local-first, study-first, and easy to run.

## Current Focus

- Open-source readiness and repo hygiene
- Strong CI and deploy confidence
- Privacy-safe docs for backups and optional AI
- Small UI polish that makes Today, SAT, and College workflows feel coherent

## Good First Issues

- Improve copy clarity in README and docs
- Add small accessibility tests around keyboard navigation
- Add screenshots or a short demo GIF to the README
- Expand documentation for curriculum authoring
- Add tests for Settings backup/import edge cases

## Larger Work

- Cross-tab storage refresh so multiple open tabs stay in sync
- More page-level integration tests for SAT and College flows
- PWA two-build update e2e test
- Print-summary layout polish for college application artifacts
- Mastery view readability polish

## Explicit Non-Goals For Now

- Backend sync or accounts
- Paid subscriptions
- ED/EA/RD-specific schema expansion
- Bulk AI-generated SAT question import
- Study intent directly changing Daily 5 question selection
- Mobile redesign beyond maintaining usability
